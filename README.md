# Telex

Python implementation of a Reddit GUI client for Linux, Windows and macOS operating systems.

## Dependencies

- gtk4 >= 4.5
- libadwaita >= 1.1
- python >= 3.11
- requests
- asyncio

## Install from source

```bash
git clone --single-branch -b main https://github.com/believemanasseh/telex-python
cd telex-python
chmod +x build.sh
./build.sh
```
